# VRCSharp
A C# asynchronous wrapper for VRChat's API

# CURRENT FEATURES
Login to accounts via username and password <br />
(Un)Friend users <br />
VoteKick users <br />
Fetch Users <br />
Fetch Worlds <br />

# ROAD MAP (Planned Features)
Mute Users <br />
Message Users <br />
Block Users <br />
Hide Users <br />
Events such as for onMuteGiven, etc <br />
Send Invites to Users <br />
